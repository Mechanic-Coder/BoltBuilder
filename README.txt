How To use:
this is a 3d interactive rendering of a hex head bolt.
the user maydownload the bolt as a gtfl file, it is reccomended to convert the file to stl useing blender if you with to 3d print the file.
the user can modify the outer diameter, thread pitch, length along with material
when the material or outer diameter is changed the user will see the tensile strength is changed
the tensile strength is displayed to a progress bar and in the small text below the bar
every time the dimensions are changed the update button must be clicked so the bolt can render


How bolt is made:
the bolt is completely rendered in all javascript in side the browser.
there is no stl file. just vertices being created inside for loops by geometric equations.
